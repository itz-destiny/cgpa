# **App Name**: GradeRight

## Core Features:

- User Interface: Develop a friendly UI for managing student data, grades, course units, and semester information.
- Automated Grading: Automate the calculation of GPA and CGPA based on student records. Ensure automatic saving to the database.
- Real-time Results: Display real-time GPA and CGPA results as grades are entered and updated.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) to promote a sense of trust and clarity.
- Background color: Very light grey (#F0F0F0) for a clean and unobtrusive backdrop.
- Accent color: A calming teal (#70A1AF) to provide visual interest and highlight key actions.
- Body and headline font: 'Inter' for a modern and readable sans-serif style.
- Use simple, outline-style icons for course management and navigation.
- Implement a clean, card-based layout to display course and grade information clearly.
- Use subtle transitions and animations to provide feedback when grades are entered or updated.
